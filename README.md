# Веб-приложение Конструктор опросников

## Функциональность

1. Возможность создания своего опросника за 2 минуты
2. Просмотр созданного опросника
3. Реализовано 6 шаблонов на проверку знаний Front-end
4. Полносью адаптивный

### Технологии

- react.js
- react-dnd
- redux
- redux-thunk
- scss
- react-Bootstrap
- html-to-image
